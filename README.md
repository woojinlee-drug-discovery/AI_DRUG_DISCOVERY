# DP1 Ligand Modeling Workflows

This repository contains machine learning workflows for DP1 ligand activity modeling and candidate prioritization.

## Overview

This repository includes two parallel modeling approaches for DP1 activity prediction:

### 1. ECFP + Random Forest Regressor

This workflow uses:

- RDKit Morgan fingerprints (ECFP-style features)
- Random forest regression
- optional classification for active/inactive labeling
- candidate scoring based on predicted `pAct`

This approach is lightweight, interpretable, and useful as a baseline cheminformatics model.

### 2. MoLFormer + XGBoost

This workflow uses:

- molecular embeddings generated by MoLFormer
- XGBoost regression
- scaffold-aware train/test splitting
- candidate scoring based on learned transformer embeddings

This approach is more representation-rich and is intended to capture broader molecular features beyond fixed fingerprints.

## Folder Descriptions

### `ECFP_RF_REGRESSOR/`

Contains workflows based on Morgan fingerprints and random forest models.

Typical tasks include:

- training a regressor on cleaned ChEMBL-style activity data
- optionally training a classifier
- scoring candidate molecules from a multi-SDF file
- exporting ranked predictions to CSV

### `MOLFORMER_XGBOOST/`

Contains workflows based on transformer-derived molecular embeddings and XGBoost models.

Typical tasks include:

- generating MoLFormer embeddings for training molecules
- training an XGBoost regressor on embedding vectors
- evaluating model performance with scaffold-aware splits
- scoring candidate molecules using the trained model
